# build

All versions.
`main.pdf` is a draft without figures.
`main.hard.pdf` is the hard-bound version.
`main.soft.pdf` is the soft-bound version.
`-j` makes all three versions in parallel.

```bash
make -j

```

Draft version only (faster).

```bash
make draft

```
